# SmartNotes

SmartNotes is a focused note-taking workspace for writing, organizing, and finding notes quickly. Zero dependencies, zero build step — just open it and type.

## Live Links

- Live app: https://rabin-portfolio-rust.vercel.app/smartnotes/
- Portfolio project page: https://rabin-portfolio-rust.vercel.app/
- Portfolio repo: https://github.com/rabinregmi188/rabin-portfolio

## Features

- Create, edit, duplicate, and delete notes
- **Markdown preview** — headings, bold, italic, lists, blockquotes, inline and fenced code, links
- **Dark / light mode** — respects your system preference, toggles manually, persists
- **Color-coded tags** with tag filtering and live search
- **Pin** important notes to keep them on top
- **Auto-save** with a status indicator (and a manual "Save Note" button too)
- **Keyboard shortcuts** — ⌘N new, ⌘S save, ⌘K focus search, ⌘/ shortcut cheatsheet
- **Word count + reading time** in the editor footer
- Export all notes as JSON
- Persists locally with `localStorage` — no backend, no sign-up

## Stack

- HTML, CSS, vanilla JavaScript
- No bundler, no dependencies — three files and a font link
- Inter + JetBrains Mono (Google Fonts)

## Run locally

Open `index.html` directly in your browser, or serve the folder with any static file server. Deploy the folder as-is to any static host (Vercel, Netlify, GitHub Pages).
