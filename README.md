# SmartNotes

SmartNotes is a focused note-taking workspace for writing, organizing, and finding notes quickly.

## Live Links

- Live app: https://rabin-portfolio-rust.vercel.app/smartnotes/
- Portfolio project page: https://rabin-portfolio-rust.vercel.app/
- Portfolio repo: https://github.com/rabinregmi188/rabin-portfolio

## Features

- Create, edit, duplicate, and delete notes
- Pin important notes to keep them at the top
- Search notes by title, content, or tags
- Filter notes by tag
- Preview formatted note content
- Export all notes as JSON
- Persist notes locally with `localStorage`

## Stack

- HTML
- CSS
- JavaScript

## Run locally

Open `index.html` in your browser.
